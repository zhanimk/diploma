# Judo Competition Automation Web Platform

This project is a web platform to automate the organization and running of judo competitions.

## Key Features
-   Athlete and coach registration
-   Tournament management
-   Application submission
-   Tournament grid generation
-   Digital refereeing
-   Automatic results update

## Tech Stack
-   **Frontend:** React, JavaScript, HTML5, CSS3, Axios
-   **Backend:** Java, Spring Boot, Spring Security, Hibernate/JPA
-   **Database:** PostgreSQL
